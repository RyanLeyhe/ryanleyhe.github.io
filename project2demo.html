How it works:

I used Pygbag to convert my asynchronous pygame code into a web-compatible format. Pygbag is a tool that enables the use of pygame in the browser by compiling it to JavaScript using tools like Pyodide and Emscripten. It's effectively Python WebAssembly that runs python code directly in the browser. After using the Pygbag build I uploaded the subsequent web files to my website's github repository so that I could link the game to this button. Just click "Ready to start!" after loading!
